# Part A — 飞书侧 UI 规范

## A.1 设计原则

| 原则 | 说明 |
|---|---|
| **信息密度优先** | 用户每日看 5-10 分钟，单卡片必须承载高密度信息 |
| **可点击下钻** | 卡片永远是"概览 + 跳转 Base 详情" |
| **数据可追溯** | 每个数字附"依据"，对应 research.md 防幻觉原则 |
| **视觉一致** | 统一颜色编码（绿=加仓/红=减仓/灰=持有/橙=止损） |
| **移动端友好** | 飞书单聊 70% 在手机查看，卡片宽度自适应 |

## A.4 多维表格视图设计

### 4.1 总体 schema

| 表名 | 用途 | 主要字段 |
|---|---|---|
| 基金池 | 全部基金 + 候选 + 观察池 | code, name, type, scale, sharpe, drawdown, reason, is_observation |
| 信号 | 每日信号记录 | date, fund_code, level, reasons, score |
| 策略竞技场 | 策略 + 排名 | strategy_id, prototype, mind_model, domain, return, sharpe, score, rank |
| 复盘 | 用户决策记录 | date, decision, fund_code, outcome, reflection |

### 4.2 视图："今日候选"（#5 fund-screener）

- **筛选：** `created_at = 今天` AND `is_observation = false`
- **排序：** 综合得分 降序
- **字段可见：** fund_code, fund_name, fund_type, scale, sharpe, reason, + "加入观察池"按钮（公式字段生成链接）

### 4.3 视图："信号"（#7 signal-engine）

- **筛选：** 用户可选"今日"/"本周"/"全部"
- **分组：** 按 level（加仓/减仓/持有/止损）
- **排序：** score 降序
- **字段：** date, fund_code, fund_name, level, reasons

### 4.4 视图："竞技场 Top-5"（#8 strategy-arena）

- **筛选：** `is_qualified = true`（满 30 天纸上模拟）
- **分组：** 按 domain（8 领域）
- **排序：** 组内 composite_score 降序
- **限制：** 每组只显示前 5（用视图筛选公式）
- **字段：** strategy_id, prototype, mind_model, return_backtest, return_forward, sharpe, drawdown, composite_score, rank

# Part B — Mac Web 前端规范（v0.3 新增）

> 对应 [context.md US-7~10](../docs/context.md) · [PRD F10/F11](../docs/prd.md) · specs/#10/#11

## B.1 设计原则

| 原则 | 说明 |
|---|---|
| **现代 SaaS 风** | 类 Notion/Linear/Stripe：干净中性、阅读友好、不追求黑色专业金融感 |
| **桌面优先** | 1280+ 显示器优先设计，平板可用，不做手机适配 |
| **类型安全** | Pydantic → OpenAPI → openapi-typescript 全链路自动生成 |
| **数据可视化优先** | 周期分析是核心价值，图表性能 ≤ 1s |
| **操作即时反馈** | 加观察池/采用策略/保存配置要乐观更新 + Toast |
| **图表可下钻** | 点击柱子/曲线点能看明细 |

## B.3 周期分析图表设计（US-7 核心）

### B.3.1 多周期收益柱状图（PeriodReturnChart）

```
        日  │█████ │ +0.85%
  收益   周  │███████████ │ +2.3%
        月  │████████████████ │ +5.1%
        季  │██████████████████████ │ +8.2%
        年  │████████████████████████████████ │ +15.5%

  [日][周][月][季][年]  ← 周期切换 tab
```

**交互：**
- 周期切换：tab 切换日/周/月/季/年，柱子数量自适应
- 鼠标悬停：显示该周期起止日 + 累计收益 + 最大回撤
- 点击下钻：点击某月柱子 → 弹出该月明细（按日拆分）
- 基准对比：可叠加沪深300同周期收益（半透明柱）

**数据契约：** `GET /api/strategies/{id}/timeseries?period=monthly`

### B.3.2 净值曲线 + 回撤 + 基准（NavCurveChart）

```
2.0 ┤                          ╭───╮ 策略净值
    │                    ╭───╯     ╰╮
1.5 ┤              ╭───╯            │
    │        ╭───╯   ╲              │
1.0 ┤──╭───╯          ╲──────╮     │
    │  ╲  策略           ╲   ╲    │
0.95┤   ╲                 ╲   ╲   │  ← 回撤阴影（灰色）
    │    ╲                 ╲   ╲  │
0.9 ┤     ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  ← 沪深300基准（虚线）
    └────────────────────────────────→
    2021              2023          2025
```

**关键配置（ECharts）：**
- 双 series：策略净值（实线）+ 基准（虚线）
- 回撤用 markArea 灰色阴影标注
- dataSampling: 'lttb' 大数据降采样（5 年 ~1200 点 → 渲染 ≤ 1s）
- dataZoom：支持时间范围缩放
- tooltip：显示日期 + 净值 + 回撤 + 基准 + 超额收益

**数据契约：** `GET /api/strategies/{id}/nav?start=&end=&benchmark=000300`
