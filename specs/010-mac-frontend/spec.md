# Feature #10 — mac-frontend（Mac 端 Web 前端）

> **Spec-Kit specify 产出** | v0.3 新增
> 来源：[PRD §6 F10](../docs/prd.md) · [context.md US-7~10](../docs/context.md)

---

## 1. Feature 简介

基于 Next.js 的 Mac 端 Web 前端，浏览器访问 `localhost:3000`，提供 5 个核心页面：策略池总览、策略周期分析、单基金深度解读、各领域推荐、完整管理台。**视觉风格：现代 SaaS**（类 Notion/Linear/Stripe）。

---

## 2. 用户故事

见 [context.md US-7 ~ US-10](../docs/context.md)。摘要：
- US-7：策略池总览 + 周期分析
- US-8：单基金深度解读（6+ 可视化模块）
- US-9：各领域基金推荐（8 领域 × Top-5）
- US-10：完整管理台（YAML 表单/观察池/策略/手动触发/版本回滚）

---

## 3. 输入 / 输出

### 输入
- FastAPI 后端 REST API（来自 #11）
- OpenAPI 自动生成的 TS 类型

### 输出
- 5 个可交互 Web 页面
- 现代 SaaS 风视觉

---

## 4. 功能需求（FR）

### FR-1：页面路由

| 路由 | 页面 | 对应 US |
|---|---|---|
| `/` | 首页 Dashboard（今日总览） | 综合 |
| `/strategies` | 策略池总览（8 领域 × Top-5） | US-7 |
| `/strategies/[id]` | 策略详情 + 周期分析 | US-7 |
| `/funds/[code]` | 单基金深度解读 | US-8 |
| `/discover` | 发现 / 各领域推荐 | US-9 |
| `/manage` | 管理台 | US-10 |
| `/manage/config` | YAML 规则编辑 | US-10 |
| `/manage/observation` | 观察池管理 | US-10 |
| `/manage/jobs` | 任务监控 | US-10 |

### FR-2：首页 Dashboard
- 今日组合表现卡片（观察池平均收益/异动 Top-3）
- 今日信号列表（加仓/减仓/止损分组）
- 今日候选 Top-5
- 竞技场亮点 Top-1
- 数据新鲜度指示器

### FR-3：策略池总览页（US-7）
- 按 8 领域 tab 切换（价值/成长/红利/趋势/逆向/全球配置/指数增强/低波）
- 每领域显示 Top-5 卡片 + 全部策略表格
- 排序：综合得分/收益/夏普/回撤
- 筛选：是否已采用、是否合格（≥30 天纸上）
- 点击策略 → 跳 `/strategies/[id]`

### FR-4：策略详情 / 周期分析页（US-7）
- 策略基本信息卡（来源原型 + 心智模型 + 参数差异说明）
- **多周期收益柱状图**（日/周/月/季/年可切换）
- **净值曲线 + 回撤阴影 + 基准对比**（ECharts 主图）
- **现金流时序图**（份额变动 + 机构持有变化）
- 时间范围筛选：近 1 月/3 月/6 月/1 年/3 年/全部
- 操作按钮：采用策略 / 查看持仓明细

### FR-5：单基金深度解读页（US-8）
- **经理画像时间轴**（任期/历史业绩/管理规模变化）
- **持仓行业雷达图**（前 N 行业占比）
- **业绩归因对比表**（本基金 / 同类平均 / 沪深300，3 列）
- **风险指标仪表盘**（夏普/回撤/波动率，仪表盘组件）
- **现金流瀑布图**（季度份额变动）
- **LLM 报告**（含【依据】可点击跳转 + 可折叠 JSON 原文）
- 操作：加入观察池 / 触发 AI 重新分析

### FR-6：发现 / 各领域推荐页（US-9）
- 8 领域分组卡片
- 每领域 Top-5 推荐基金
- 每只基金附推荐理由（命中规则 + 指标值）
- 时间窗切换：今日 / 本周 / 本月
- 一键加入观察池

### FR-7：管理台（US-10）
- **YAML 规则表单化编辑**（不用手写 YAML，结构化表单）
- **影响范围预览**（保存前显示"影响 N 只基金"）
- **观察池管理**（列表 + 增删 + 排序）
- **策略管理**（已采用列表 + 停用按钮）
- **手动触发**（拉数据/跑回测/生成报告，进度条）
- **配置版本历史**（git log 列表 + 一键回滚）

### FR-8：交互与状态
- 全局 Toast 通知（操作成功/失败）
- 操作确认 Modal（删除/回滚）
- Loading 骨架屏（数据加载）
- 错误边界（Error Boundary）

## 5. 非功能需求（NFR）

| 维度 | 要求 |
|---|---|
| 首屏加载 | ≤ 2s（本地） |
| 图表渲染 | ≤ 1s（5 年数据） |
| 响应式 | 桌面 1280+ 优先，平板可用 |
| 类型安全 | 全链路 TS 类型（openapi-typescript） |
| 视觉一致 | shadcn/ui 组件库统一风格 |

## 6. 验收标准（AC）

### AC-1：策略池总览（US-7）
**Given** 进入 `/strategies`
**When** 切换"价值"领域
**Then**
1. 显示价值领域 Top-5 策略卡片
2. 下方表格列全部价值策略
3. 点击策略跳详情页

### AC-2：周期分析图表（US-7）
**Given** 策略详情页
**When** 切换"月度"周期
**Then**
1. 柱状图渲染月度收益
2. 含基准对比
3. 点击某月柱子显示明细
4. ≤ 1s 渲染

### AC-3：单基金深度解读（US-8）
**Given** 进入 `/funds/000001`
**When** 页面加载
**Then**
1. 6 个可视化模块全部渲染
2. 业绩归因含 3 列对比
3. LLM 报告【依据】可点击
4. "触发 AI 重新分析"按钮可用

### AC-4：发现页（US-9）
**Given** 进入 `/discover`
**When** 切换"本月"
**Then**
1. 8 领域 × Top-5 列表
2. 每基金附理由
3. 一键加观察池成功

### AC-5：管理台（US-10）
**Given** 进入 `/manage/config`
**When** 修改规则并保存
**Then**
1. 显示影响范围（"影响 N 只"）
2. 自动 git commit
3. 历史列表可见
4. 一键回滚生效

### AC-6：类型安全
**Given** OpenAPI 类型已生成
**When** 前端编译
**Then** 无 TS 类型错误

## 7. 显式不做

- ❌ 不做认证 / 登录（本地单用户）
- ❌ 不做移动端适配（桌面优先）
- ❌ 不做 SSR（纯 CSR 即可，本地访问）
- ❌ 不做国际化（中文）
- ❌ 不做暗色模式（MVP 仅亮色）
